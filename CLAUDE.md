people outside this session — this includes (but is not limited to):

- Branch names (do not use a `claude/...` prefix or similar)
- Git commit author/committer identity
- PR/issue titles, bodies, and comments (no "Generated by Claude Code"

- git author/committer: `DJAKN <18362174+DJAKN@users.noreply.github.com>`
- Branch names: use plain descriptive names (e.g. `dev/...`,

are fine as-is — this rule is specifically about what becomes visible to
external people on GitHub or elsewhere.
